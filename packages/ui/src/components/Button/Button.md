Default example:

```js
<div>
    <Button>
        Hello
    </Button>

    <Button accent>
        Accent
    </Button>

    <Button raised accent>
        Accent and Raised
    </Button>
</div>
```

Using router-link example:

```js
<div>
    <Button to="/">
        Linker
    </Button>

    <Button raised to="/">
        Link raised
    </Button>

    <Button accent to="/">
        Link accent
    </Button>

    <Button raised accent to="/">
        Link accent and raised
    </Button>
</div>
```
